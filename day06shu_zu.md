# day06-数组

数组就是存储一组相同数据类型的数据的集合。

## 一维数组的定义和使用
数组就是在内存中连续的相同类型的变量空间。

    int main()
    {
        /*
        这句代码定义了5个int类型的变量。
        第1个变量名是arr[0],
        第2个变量名是arr[1],
        ...
        最后的一个变量名是arr[4],
        */
        int arr[5];
        
        return 0;
    }

注意：
* 定义数组时，[]里必须是常量。使用数组时，可以是常量，也可以是变量。
* 数组的下标的取值范围：0 ~ 数组长度-1
