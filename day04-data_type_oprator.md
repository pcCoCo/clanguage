## 运算符

* ()改变优先级

* 强制转换(显式转换) 
 
* 隐式转换
    类型转换
        3/2 = 1 ---> 操作数都是整数，结果也是整数。
        3/2.0 = 1.5 ---> 操作数类型不同，运算之前会先将精度低的操作数隐式转换为精度高的数据类型，再运算，表达式的结果是高精度的数据类型。
        
// 隐式转换例子

```
int main()
{
    unsigned int a = 99;
    unsigned int b = 100;
    if(a-b < 0)
    {
        printf("%u\n", a);
    }
    else
    {
        printf("b\n");
    }
    return 0;
}    
```
